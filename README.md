# 逆説的AGI / Paradoxical AGI

このリポジトリは、AIシステムの実装や完成形を提示するものではありません。
また、特定の技術的アーキテクチャや倫理的立場を主張するものでもありません。

## これは何ではないか

- AGIを定義する試みではありません
- 新しいAI製品・フレームワークの提案ではありません
- 人間の判断を代行する思想ではありません
- 社会的な正解や指針を示すものではありません

## 読み方について

このリポジトリに含まれる文書は、
「理解すること」や「合意すること」を目的として書かれていません。

- 読み切る必要はありません
- 正しい解釈は存在しません
- 途中で読むのをやめても問題ありません

## それでも、ここに置かれている理由

AIが高度化し、判断や決定が外部化されやすくなる中で、
「判断がどこにも回収されずに残る配置」が
一つの可能性として存在していた、という痕跡を残すためです。

## 構成について

- `/canon/`  
  正典文書群。逆説的AGIおよび知的生態系に関する、構造的・配置的な記述です。

- `/docs/`  
  補足・観測・周辺文書。理解を補助することを目的としていません。

※ 正典は重く、読解を要求します。必要だと感じた場合のみ参照してください。
